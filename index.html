Skills

Over the past year, I've been on a bit of a rollercoaster ride, with minimal coding experience (some HTML and CSS around 20 years ago); it's been a brilliant journey so far.
A little confused at where to start, I began learning Python with the intention of becoming a Software Engineer but as part of this a course I was doing involved making a small web app and I was hooked instantly. Turns out Web Development is the career for me!
Since then, I have undertaken multiple courses and challenges trying to absorb everything Front and Back-end that I can. So far I have started learning: HTML, CSS, JavaScript, PHP and some Node.js. Alongside this I have also used Frameworks including Bootstrap, Jquery and ExpressJS.
Of course, there's no point becoming a Developer without ditching the Windows GUI so I've tried to master the Command Line Interface, as well as getting familiar with version control using both Git and GitHub.

JavaScript Dice Game

Game of Rock, Scissors, Paper


This was a project I decided to do to show off some JavaScript skills as well as use of media breakpoints.
For this to work I created a random number generator for each player, I then assigned Rock, Paper and Scissors with the numbers 1, 2 and 3 respectively. Next, I used some logic to determin who won, ie, 1 (Rock) beats 3 (Scissors). I used Javascript to change the image displayed on-screen and change the text to say who won.
